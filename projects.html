Real Time Bus Tracker

The Real Time Bus Tracker project tracks bus movement in real time (at intervals of every 15 seconds) throughout part of the city. This project employs a public API key, in harmony with the Google Maps cloud platform, to incorporate live data to track movement across an animated map.
